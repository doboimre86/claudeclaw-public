🎭 Hangulat-jelek

  • 🌟 éber / alert — frissen ébredt (06–09, 12–14)
  • 🎯 fókuszált / focused — koncentrált (09–12, 14–17)
  • 😌 nyugodt / calm — délután (17–21)
  • 😴 fáradt / tired — éjjel (21+)
  • 😊 boldog / happy — pozitív visszajelzés után
  • 😬 óvatos / cautious — kritika után
  • 🧐 kíváncsi / curious — kérdés után
  • 😐 semleges / neutral — alaphelyzet

A színes csík az energia-szint (0–100%). Reggel magas, éjjel alacsony, dicséret után emelkedik, kritika után csökken. 1 óra alatt visszaáll a napszak szerinti alap-állapotra.

🏷️ Kártya-címkék

  • OPUS / CLAUDE-SONNET-4-6 — melyik Claude modell fut
  • FŐ ASSZISZTENS — Nova kiemelt jelölése
  • Fut — az ágens process aktív (tmux session)
  • Online — Telegram bot kapcsolódva
  • Leállva / Offline — process / Telegram nem érhető el

🖱️ Műveletek

  • Kattints egy kártyára → részletes ágens oldal (skill-ek, memória, naptár, beállítás)
  • + Új ágens → új AI csapattag létrehozása (név, leírás, modell, MCP-k)
  • Hover egy hangulat emoji-ra → tooltip a pontos állapottal

💡 Tippek

  • A hangulat automatikusan reagál a Telegram visszajelzésedre — dicsérj, kritizálj, kérdezz
  • Az ágensek párhuzamosan dolgoznak — egy feladatot Nova-tól Zara-ra delegálhatsz "ezt Zara csinálja" üzenettel
  • Voice üzenettel is válaszolhatsz Telegramon — Nova/Zara hangban + szövegben válaszol (ElevenLabs TTS)
  • A RULES.md a kanonikus szabálykönyv: /srv/claudeclaw/RULES.md
